# Beckett Pizza Plaza - 4-for-3 Pizza Offer

## Description
This is a Python program to calculate the price of a pizza order at Beckett Pizza Plaza.  
The offer is “Four-for-Three”: when a customer orders four pizzas, the cheapest pizza is free.  
The program will calculate the order total and show the discount achieved.

## Features
- Accepts four pizza prices from the user
- Validates input (no negative or invalid prices)
- Calculates total and discount percentage
- Optional: can read prices from a file (4 prices, one per line)
- Can run from the command line

## Files
- `pizza_offer.py` → main program
- `sample_prices.txt` → optional file containing pizza prices
- `requirements.txt` → list of dependencies (empty if none)
- `README.md` → this file

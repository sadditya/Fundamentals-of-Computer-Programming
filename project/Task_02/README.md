# Password Security Checker - Task 2

## Description
This Python program verifies user passwords with the following rules:
1. Password must be at least 9 characters long.
2. The program asks the user for 3 letters at random positions.
3. If all letters are correct, security check passes; otherwise, it fails.

Optional: program can read passwords from a file (one password per line).

## Files
- `password_checker.py` → main program
- `sample_passwords.txt` → optional file for testing passwords
- `requirements.txt` → empty (no external libraries)
- `README.md` → this file

1. Manual password entry:
```bash
python password_checker.py
